# mit-6.s894-accelerated-computing
### Overview
A self-guided run-through of the MIT accelerated computing course. I have taken software performance engineering and I crave more. This is not offer Fall 2026 so I plan to complete this entire course on my own during the summer.
### Repository Setup
##### Labs as Submodules
Each of the labs were added to this repository as a submodule using this command:
```
git submodule add <URL_OF_REPO_TO_CLONE>
```
To update labs into a local repository, run this command after cloning and getting all latest changes: 
```
git submodule update --init --recursive
```
